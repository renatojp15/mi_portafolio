Mis Proyectos

Proyecto E-commerce

Un sistema completo de comercio electrónico que incluye funcionalidades como gestión de usuarios, productos, carrito de compras y un sistema de autenticación. Diseñado para proporcionar una experiencia profesional tanto para los usuarios como para los administradores.

Tecnologías utilizadas: Node.js, JavaScript, MySQL, EJS(motor de plantillas), React, Sequelize, CSS

Una interfaz de administración moderna diseñada para mostrar métricas y estadísticas clave. Este proyecto presenta paneles dinámicos que se conectan con una API para obtener datos en tiempo real y ofrecer una experiencia interactiva y visualmente atractiva.

Tecnologías utilizadas: Node.js, JavaScript, React, CSS
